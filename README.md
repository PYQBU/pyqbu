# BU PYQ - Bhattadev University Previous Year Question Papers

A modern, responsive web application for accessing previous year question papers from Bhattadev University.

Built with React, Vite, and Tailwind CSS for optimal performance and user experience.

## Features

- 🔍 Instant search functionality
- 📱 Mobile-first responsive design
- 🌙 Dark mode toggle
- ⚡ Lightning-fast performance with Vite
- 🎨 Beautiful modern UI with Tailwind CSS
- 📄 PDF previews and downloads
- 🎓 Department and semester-wise organization
- 🔄 Lazy loading for optimal performance
- ♿ Full accessibility support
- 📱 PWA support

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Axios

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deploy to Netlify with one click or use:

```bash
npm run build
```

Then deploy the `dist` folder to Netlify.
